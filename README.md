# README
ビリヤード動画まとめサイトポートフォリオ。
下記に実装済み機能、未実装機能を記す

## 開発環境
- Ruby 2.4.1
- rails 5.2.3
RDBMS
Production
- postgresql 11.4
development
- sqlite3 1.3.13
Web Server
- puma 3.12.1


## 機能
- ユーザのCURD機能(新規作成、編集、削除、一覧、検索)
- ユーザ一覧のページネーション機能(kaminari)
- 外部APIによる動画情報取得機能(google-api-client)
- 動画タイトル一覧表示機能
- 動画サムネイル表示機能
- 動画ページリンク機能
- 動画コメントのCURD機能(新規作成、編集、削除、一覧)
- ログイン・ログアウト機能
- ログイン状況に応じた表示切替
- テスト機能(一部実装)
